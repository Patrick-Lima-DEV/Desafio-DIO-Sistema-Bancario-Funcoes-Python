VERSÃO 2.0 (Base - 5 Melhorias Principais)
──────────────────────────────────────────

1. ✓ PERSISTÊNCIA EM JSON
   - Dados salvos automaticamente em "dados_bancarios.json"
   - Carregamento ao iniciar o programa
   - Funções: salvar_dados(), carregar_dados()

2. ✓ VALIDAÇÕES
   - CPF com algoritmo verificador MOD 11
   - Data no formato dd-mm-aaaa
   - Funções: validar_cpf(), validar_data()

3. ✓ TESTES UNITÁRIOS
   - 21 testes com pytest
   - test_sistema_bancario.py
   - Cobertura: validações, usuários, contas, transferências

4. ✓ TRANSFERÊNCIA ENTRE CONTAS
   - Nova funcionalidade operacional
   - Opção [t] no menu
   - Registra com timestamp em ambas as contas

5. ✓ LOGS COM TIMESTAMPS
   - Formato: [dd/mm/aaaa HH:MM:SS]
   - Em: depósito, saque, transferência
   - Auditoria completa

VERSÃO 2.1 (Adicional - Melhoria de UX)
───────────────────────────────────────

6. ✓ LOOPS DE VALIDAÇÃO
   - criar_usuario() - CPF e data em loops
   - criar_conta() - CPF em loop
   - selecionar_conta() - número em loop
   - depositar() - valor em loop
   - sacar() - valor em loop
   - transferir() - valor em loop
   - Benefício: dados não são perdidos em erro de validação
